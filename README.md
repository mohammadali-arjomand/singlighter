# Singlighter
Singlighter is an installer/assistant for SinglightJs framework

# Directories
Singlighter will making directories like it
```
SinglightJs
├── Scripts
│   ├── Pages
│   │   └── HomePage.js
│   ├── Router.js
│   ├── App.js
│   └── Singlight.js
├── Styles
│   └── main.css
└── index.html
```
